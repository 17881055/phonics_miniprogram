# 欢迎使用 Phonics 小程序


** Phonics 小程序 本来是为我女儿复习 Phonics 自然拼读 和 复习 Sight word 而开发的**

## 目前功能
* Phonics （ 自然拼读 ） ，包含 26，46，54个字母的玩法。
* Sight Word （ 高频词汇 ）有声跟读。 现在是 94个高频词汇 。
* 填字母游戏。
